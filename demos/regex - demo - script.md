# Taming the dragon - An introduction to Regular Expressions - Demos!

## Input Validation

### 'Our serial numbers always have 6 hex-digits'
Possible solution:

    ^[0-9a-zA-Z]{6}$

### 'Names are made of two parts with a space in between'
Possible solution:

    ^(\S+) (\S+)$

Notes:
- Names are not all composed of two parts. Probaly it is 1 to many
- `\S` is probably too much, but remember there are many languages having non-english/-latin letters like á, ö, ß, я, … and people will have those in names. Also: people travel and move.

### '`a` is repeated an "unprime" number of times'
Possible solution: https://regex101.com/r/152Ua4/3

    ^(a{2,}?)\1++$

Notes:
- Probaly not a too common case, but nice to show a bit of the flexibility
- Way more difficult: only validate length without assuming anything about the content

## Editor Demos

### Find all usages of an object in a script (quoted, default schema, …)

Possible solution:

```
(?<!\w)(?:\[dbo\]\.|dbo\.|(?<!\.))(?:\[Tbl\]|Tbl)(?!\w)
```

A bit less accurate:

```
\b(?:\[?dbo\]?\.|(?<!\.))(?:\[?Tbl\]?)\b
```

### Rename procedures: CREATE OR ALTER replaced by DROP IF EXISTS + CREATE OR ALTER

Possible solution:

```
CREATE(?: OR ALTER|  )? PROCEDURE ((\S+)\.(\[)?sp_(\S+\]?))
```
```
DROP PROCEDURE IF EXISTS $1;
GO
CREATE OR ALTER PROCEDURE $2.$3$4
```

### Rewrite UPDATE to INSERT: UPDATE x SET y = ? replaced by INSERT x (y) VALUES (?)

Possible solution:

```
UPDATE V
SET VAL = (\S+)
FROM our\.Vault V
WHERE V\.id = (\S+);
```
```
INSERT INTO our.Vault (Id, Val)
VALUES ($2, $1);
```

## Regular Expression? Let's just parse.

### Normalize line-endings

So you may have CR LF, just LF or even just CR? Let's normalize to CR LF!
- SQL:
```sql
SELECT REPLACE(
        REPLACE(
            REPLACE('a' + CHAR(13) + CHAR(10) --
                    +'b' + CHAR(10) --
                    +'c' + CHAR(13),
                CHAR(13)+CHAR(10),
                CHAR(10)),
            CHAR(13),
            CHAR(10)),
        CHAR(10),
        CHAR(13)+CHAR(10));
```
- C# `Replace`s:
```cs
"a\r\nb\nc\r"
    .Replace("\r\n", "\n")
    .Replace("\r", "\n")
    .Replace("\n", "\r\n");
```
- C# loop:
```cs
public static string NormalizeLineEndingsLoop(IEnumerable<char> input)
{
    var output = new StringBuilder();
    var lastWasCR = false;
    foreach (var c in input)
    {
        if (lastWasCR)
        {
            output.Append("\r\n");
            switch (c)
            {
                case '\r':
                    break;
                case '\n':
                    lastWasCR = false;
                    break;
                default:
                    output.Append(c);
                    lastWasCR = false;
                    break;
            }
        }
        else
        {
            switch (c)
            {
                case '\r':
                    lastWasCR = true;
                    break;
                case '\n':
                    output.Append("\r\n");
                    break;
                default:
                    output.Append(c);
                    break;
            }
        }
    }
    
    if (lastWasCR)
    {
        output.Append("\r\n");
    }

    return output.ToString();
}
```

```
\r?\n|\r
(?<!\r)\n|\r(?!\n)
```
and replace with
```
\r\n
```
